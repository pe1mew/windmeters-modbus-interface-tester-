Wind Interface

Device/system registers (identity + bus health) for the DUT itself — identical on every build (TDS §2.7). These are the DUT SLAVE's own counters (0x0008/0x0009, Modicon 30009/30010) — distinct from the tester's OWN Modbus bus health shown in the "Modbus Bus" status card above.

Reads once on demand — and also updates automatically whenever a Wind Speed / Direction / Combined poll is running, since these registers ride along in the same read.
